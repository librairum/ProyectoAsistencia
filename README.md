PASOS PARA CORRER SISTEMA:
1. Cambiar datos en server.js en donde se encuentra la bd de Asistencia
2. Ahora para correr el sistema solo se debe escribir en la consola "node server.js"
3. Caso contrario no deja iniciar se debe habilitar la configuracion del SQLSERVER y reiniciar el servicio de SQL SERVER (MSSQLSERVER)
